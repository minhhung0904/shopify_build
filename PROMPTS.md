# Shopify AI Store — All Prompts

## PROMPT 01 — VERIFY TOOLKIT WORKING
Search the Shopify documentation and tell me what Liquid variables are available on a product page template. Show me live documentation, not from memory.

## PROMPT 02 — BUILD A PRODUCT PAGE
Build me a high-converting product page for my Shopify store. Product: [name]. Features: [list 3-5]. Target customer: [describe]. Include: trust badges (reviews, secure checkout, free shipping, money-back), urgency element, strong above-the-fold section. Show me the Liquid code.

## PROMPT 03 — CLONE A COMPETITOR LAYOUT
Analyse the conversion structure of [competitor URL] — section order, trust elements, urgency triggers, CTA placement. Build me a version of this layout for Shopify using Liquid, adapted for my brand [name]. Don't copy their text — just the layout and conversion structure.

## PROMPT 04 — AI PRODUCT DESCRIPTION
Write a high-converting product description for Shopify for: [product name and description]. Target audience: [describe]. Format: headline, 3-4 short paragraphs, bullet benefits list (6-8 bullets), closing CTA. Tone: [premium/casual/urgent]. Make it feel like a direct-response landing page.

## PROMPT 05 — FIX MOBILE LAYOUT
My Shopify product page has this mobile issue: [describe issue]. Show me the CSS and Liquid code to fix it and make it fully mobile responsive. Target breakpoint: 768px and below.

## PROMPT 06 — ADD TRUST BADGES
Add trust badges below the Add to Cart button on my product page: "Free Shipping", "30-Day Money Back Guarantee", "Secure Checkout", "4.8 Star Reviews". Use icons. Write the Liquid and CSS code I need to add to my theme.

## PROMPT 07 — FULL STORE BUILD
Help me build a complete branded dropshipping store on Shopify for [niche]. I need: homepage (hero, featured products, brand story), collection page, high-converting product template, About Us page, Contact page, and navigation structure. Brand tone: [premium/casual/energetic]. Target customer: [describe]. Give me the plan first, then build section by section.

## PROMPT 08 — CRO AUDIT
Review my Shopify product page and identify the top 5 conversion rate optimisation improvements. Look for: missing trust signals, weak headline, unclear CTA, poor mobile layout, missing urgency, weak social proof. For each issue, tell me what to change and provide the code to fix it.
